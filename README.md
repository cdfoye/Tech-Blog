  ![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

  ## Description

   This is a CMS-style blog site where developers can publish their blog posts and comment on other developersâ€™ posts as well! This app follows the MVC paradigm in its architectural structure, uses Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication. This application is deployed using Heroku.
